About this service

Get adult social care data is a digital insight tool that provides access to a wide range of datasets relevant to adult social care in England.

The service brings together data from the Care Quality Commission, Office for National Statistics, NHS England, Capacity Tracker and other sources. Learn more about the data sources.

When new data is published, the service updates automatically so you can get insights based on the most current information.

You can use this service to:

  • access data and track indicators across a range of topics, using the latest available information
  • explore and visualise data with interactive tools that help you to identify patterns, trends and insights
  • support decision-making and strategic planning by downloading datasets and charts in formats you can integrate into your own analyses

Help with finding data

There are different ways to find data in this service:

You can search across indicators and topics to find specific information. Search for data now.

If you prefer to explore related data, you can browse data by topic.

On topic pages, you can adjust the data shown using the Change your data preferences option.

To access more detailed information about an indicator, use the Explore more data option on a topic page.

If you need the raw data to explore with your own analytical tools, you can download full datasets.

Data accuracy and completeness

The data in this service is sourced from established organisations. However, each dataset has been collected using different methods and may have gaps or variations in how data is recorded.

What to consider when using data

When using the data, you should be aware of:

  • coverage – some datasets may not include all local authorities or demographic groups due to data availability.
  • methodology differences – the way data is collected and categorised can vary across sources, which affects direct comparisons.
  • timeliness – some datasets are updated more often than others, so there may be a delay between data collection and publication.
  • confidence levels – some indicators are based on sample data or estimates rather than full counts, so figures should be treated as estimates.

Where to find more details

Each indicator includes footnotes explaining the limitations and methodology of each dataset used. These are highlighted on the relevant indicator's page.

You can also check the official documentation from the data provider, where available.

Data sources

Find out where the data is sourced from, including details of data providers and how updates are managed.

Get adult social care data uses the following public and private data sources.

Public data sources

  • Adult Social Care Statistics (Department for Health and Social Care)
  • Adult Social Care Outcomes Framework (ASCOF) (NHS England)
  • Adult Social Care Report (Department for Health and Social Care)
  • Care Quality Commission (CQC) Care Directory
  • Office for National Statistics collection
  • Short and Long Term Return (SALT) (NHS England)

Private data sources

  • Capacity Tracker
  • Skills for Care
  • Client Level Data (CLD)

Data sharing and re-use

Unless stated otherwise, you may re-use this information (excluding logos) free of charge in any format or medium, under the terms of the Open Government Licence v.3.0 (opens in new tab).

If we have identified any third-party copyright information, you must get permission from the copyright holders to re-use it.

! Warning Any data from Capacity Tracker in this service is for private use only and must not be shared or published externally. By accessing this service, you agree to these conditions.

How we use artificial intelligence (AI) in this service

Learn how AI is used in this service, including an overview of Smart insights, how it works, its limitations, and best practices for using it effectively.

Overview

Smart insights is a feature that uses AI to help analyse data and identify key trends and patterns.

This is designed to support - but not replace - your own analysis of the data.

As it's experimental, there are some limitations to keep in mind.

How it works

Smart insights uses AI models trained on large datasets to identify relationships within the data.

These models generate summaries and interpretations designed to simplify complex information and help you uncover useful insights.

Limitations

AI-generated insights are generally reliable, but may not always be complete, reflect the most recent policy changes, or capture important context.

Sometimes, AI might misinterpret data, producing insights that do not fully align with the underlying trends or patterns.

Best practices

Always cross-check the AI-generated insights with other data sources to get a fuller picture.

Review the insights carefully, and verify they are correct and appropriate for your needs before sharing or using them.
