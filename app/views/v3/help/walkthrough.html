Step-by-step walkthrough

Learn how to navigate the service and use its features with step-by-step instructions.

1. Browse topics

  • Go to the Get data page.
  • Explore the available topics, such as Current demand and provision.
  • Select a topic to view related data, insights and key indicators.

Try this now

2. View and explore data

  • On a topic page, choose the section you want to explore further.
  • You can find an overview and key highlights relevant to your organisation.
  • Select Explore data to access a more detailed breakdown.

Try this now

3. Refine and customise data views

  • On the detailed view pages, you can use the available filters to refine the data further.
  • You can also change how the data is displayed by selecting a chart type, such as bar chart, line chart or map.
  • Select Apply changes to update your view.

Try this now

4. Access insights and supporting information

  • On topic pages, you can find Smart insights, which use artificial intelligence (AI) to analyse the data and highlight trends and patterns.
  • To understand how the data is collected and what it represents, select View supporting information for the data.

Try this now

5. Get help using the service

  • If you need further support or guidance, you can view more Help topics.
